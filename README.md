# Coop Bank App

Banking app using React Native.
This app includes: 
- Home
- Login
- Logout
- Forgot password
- Help center page
- Accounts page
- Drawer navigation

## Prerequisites
- NodeJs
- Expo CLI
- npm or yarn

## Setup
#### Clone repo
`https://github.com/dluismendezpy/coop-bank-app.git`

#### Install dependencies
`npm install` or `yarn install`

#### Run app
`yarn start`

#### Run on Android/Ios device directly
`yarn android` / `yarn ios`
